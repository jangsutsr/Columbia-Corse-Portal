System Requirements

 

Acrobat 9 Pro for Mac OS® requires the following:

 

·         PowerPC® G4, G5 or  Intel® processors

·         Mac OS X version 10.4.11, 10.5

·         256 MB of installed RAM (512 MB recommended)

·         1.5 GB of available hard disk space

·         1024 x 768 screen resolution

·         DVD-ROM drive, if installing from a DVD

·         Safari 3.0.4

Installation Instructions

 

Before you can start Acrobat, you must install (copy) the Acrobat folder to your hard drive and run the software from that drive.

 

Use the following instructions to install new software or to upgrade from a previous version. When you upgrade, the Acrobat 9 folder that you copy to your hard drive contains all the files required for running the software; previous installations of Acrobat are not affected. If you purchased Acrobat as a download, Adobe recommends that you back up your download package, preferably to a DVD.

To install and start Acrobat 9 in Mac OS®

 

1. Start the installation:

·         If you purchased Acrobat on a DVD, insert the DVD in the DVD drive to view the application files.

·         If you purchased Acrobat as a download, double-click the DMG file to mount the image file as a volume.

 

2. Drag the Adobe Acrobat 9 Pro folder to the Applications folder at the root of your hard disk, open the folder, and then double-click the Adobe Acrobat Pro icon.

 

3. In the Adobe Acrobat installer dialog box, accept the license agreement, and then click Continue.

 

4. Enter the serial number for your copy of the program. The serial number is located on the outside of the DVD case, or on the receipt page for web purchases. You can also locate the serial number for web purchases in your online order history.

 

5. In the Authenticate dialog box, type the user name and password for an administrator account, and click OK. The installer lets you select optional components.

 

Note: If you don't know the name and password for an administrator account, contact the person who administers the computer.

 

6. To start Acrobat, open the Adobe Acrobat 9 Pro folder and double-click the application icon, or double-click a PDF file.

 

To uninstall Acrobat 9 in Mac OS®

 

1. Double-click the Acrobat Uninstaller file, located in the Adobe Acrobat 9 folder.

 

2. Follow the on-screen instructions to uninstall Acrobat.

 

3. In the Authenticate dialog box, type the user name and password for an administrator account and click OK.

 

Note: The uninstaller removes only Adobe Acrobat 9 files from the system.  It does not remove Adobe Reader files, Acrobat user data, or files that are shared with other Adobe applications.

 

4. (Optional) Move any Adobe PDF Settings files that you created from the Library/Application Support/Adobe PDF folder and drag the folder to the Trash.

 

Registration

 

Please complete your registration. When you start Acrobat, Acrobat may prompt you to register your software online. You may also choose Registration from the Help menu to register your software online. Alternatively, you can register your software by filling in and mailing the registration card provided with Adobe Acrobat. You must register to receive technical support.
